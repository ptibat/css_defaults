# css_defaults


**css_defaults**, permet d'appliquer des styles CSS par défaut sur une page HTML.  



## Compatibilité
CSS3


## Installation

En entête de la page HTML ( \<head> ) :
	
	<link rel='stylesheet' href='./defaults.min.css' />

Import CSS :

	@import url( "./defaults.min.css" );


That’s all
